# cordova-plugin-xinge
The cordova plugin for xg push from tencent.

```
cordova plugin add https://github.com/bqluan/cordova-plugin-xinge.git \
    --variable ANDROID_XG_ACCESS_ID=<YOUR_ACCESS_ID> \
    --variable ANDROID_XG_ACCESS_KEY=<YOUR_ACCESS_KEY> \
    --variable IOS_XG_ACCESS_ID=<YOUR_ACCESS_ID> \
    --variable IOS_XG_ACCESS_KEY=<YOUR_ACCESS_KEY>
```
